<Summary>
experimental/users/nafi/mcf_api_tutorial
Codes for demonstrating how to query the MCF API version 3 (alpha) using the
Google API Javascript Client Library. It also demonstrates visualization using
Raphael's Javascript Graphics Library.

hello_mcf_api.html
- The web page

js/main.js
- Performs API call to MCF API
- Parse data from MCF API response

js/auth_util.js
- Utility script to handle Google API Client's authorization
- User must populate API Key and Client Id in this file

js/draw.js
- Visualize parsed data from MCF API response using raphael.js

js/colors.js
- Utility script to create colors

js/util.js
- Utility classes to write inner HTML, to do simple mapping

css/pretty.css
- Style file for the web page
</Summary>
